List tags can be used to create an unordered or an ordered list.
This is an unordered list.
  • Number 1
  • Number 2
  • Number 3
This is an ordered list.
  1. Number 1
  2. Number 2
  3. Number 3
    <ul>
        <li>Number 1</li>
        <li>Number 2</li>
        <li>Number 3</li>
    </ul>

    <ol>
        <li>Number 1</li>
        <li>Number 2</li>
        <li>Number 3</li>
    </ol>
