Table tags use tr, th and td elements to print a table to the screen.
th elements represent the header. This is column 2.
td elements represent a single cell This is cell 2.
tr elements represent a row.
This is row 4 
<table>
    <tr>
        <th>th elements represent the header.</th>
    <th>This is column 2.</th>
    </tr>
    <tr>
        <td>td elements represent a single cell</td>
    <td>This is cell 2.</td>
    </tr>
    <tr>
        <td colspan="2">tr elements represent a row.</td>
    </tr>
    <tr>
        <td colspan="2">This is row 4</td>
    </tr>
</table>
